# Event-Platform
 <p> Plataforma hospedada : https://event-platform-online.vercel.app/ </p>
 <p>Uma Plataforma de evento com uma Home onde se poderá fazer o cadastro no Evento, após o submit o usuário é levado à pag Event onde podera assistir as aulas sendo elas Categorizadas em aulas Prática,Ou ao vivo, Ou se ainda não postada com tag "Em Breve".</p>
 <p> As demais funcionalidades estão funcionando, porém, sem a linkagem de teste/demonstração até a última edição deste projeto</p>
